get start with protractor

1. install node js
2. run the setup_proxy.bat file to setup proxy to download files from npm
3. run npm install protractor@2 -g
4. run webdriver-manager update
5. test if things are set up, run webdriver-manager start

if you get error on 5, then install java sdk from http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
